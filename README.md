#Git verkefni
###1.Fyrir hvað stendur skammstöfunin RAM? Útskýrðu hvað RAM er.
"Random access memory" notað sé, skammtímis vinnu grymsla fyrirm gögn og sem verið að nota við CPU.
###2.Fyrir hvað stendur skammstöfunin ROM? Útskýrðu hvað ROM er.
"Read only memory chip" og inniheldur leiðbeiningar sem getur bara leyft aðgang fyrir CPU
