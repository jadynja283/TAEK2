###1.Fyrir hvað stendur skammstöfunin RAM? Útskýrðu hvað RAM er.

###2.Fyrir hvað stendur skammstöfunin ROM? Útskýrðu hvað ROM er.
"Read only memory chip" og inniheldur leiðbeiningar sem getur bara leyft aðgang fyrir CPU

###4.Fyrir hvað stendur skammstöfunin ESD, havð er það og hvaða leiðir höfum við til að verjast því þegar unnið er með tölvubúnað?
Electrostatic discharge(ESD) can occur when there is a buildup of an electric charge(static electricity) that exsits on the surafce which comes into contact with another

ESD can cause permanent damage to electrical components
follow these reccomendations to help prevent ESD damage

*Keep all the components in anatistatic bags until you are ready to install them

*Use grounded mats on workbenches

*Use grounded floor mats in work areas

*Use anatistatic wrist straps when working on computers
