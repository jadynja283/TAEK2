#Git spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	cd = change directory, notuð til að færa sig yfir í möppu.
	cd ~ = Fer í /h drifið, fer á heimasvæði notandans.
	ls = listi working directory, sýnir möppur og skrár.
	pwd = print working database, sýnir hvar þú ert.
	mkdir = make directory, gerir nýja möppu.

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone = Afritar geymslu
	git log = Sér allar skráningar
	git status = Shows the working tree status
	git diff = shows changes on commits, commit and working tree etc
	git checkout = switches branches or restore working tree files

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff = Shows changes on commits, comit and working tree etc
	b) git diff --staged = changes between the index and your last commit
	c) git diff commit1 commit2 = compare the version between commits

4. 	Hvað er útgáfustýring (e.version control)? 
	svar = Version control er kerfi sem tekur upp breytingar í skrám svo að þú getur farið yfir þær seinna

5.	Hverjir eru helstu kostir við að nota GIT?
	svar = Það er léttara að vinna með vinnufélögum því þú getur séð þegar einhver breytir 

6.	Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu?
	svar = eftir hvert skipti sem þú gerir einhverjar breytingar.

7.	Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT?
	"Working directory" er mappa sem git er að vinna í á stundinni
	"Staging area" er mappa sem er venjukega geymd í git skránni sem geymir upplýsingar um hvað fer inn í næsta commitið þitt
	"Repository" geymir skjöl og breytingaskrár af því sem að þú breyttir í þessum skjölum

8.	Hvenær er sniðugt að nota greinar (branches)? 
	svar = þegar þú ert að gera eth nýtt eins og að laga villu er gott að gera grein því þá geturu breytt því eins mikið og þú villt

9.	Hver fann upp GIT?
	svar = Linus Torvalds árið 2005
