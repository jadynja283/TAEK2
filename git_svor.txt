#Git spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	cd = change directory, notuð til að færa sig yfir í möppu.
	cd ~ = Fer í /h drifið, fer á heimasvæði notandans.
	ls = listi working directory, sýnir möppur og skrár.
	pwd = print working database, sýnir hvar þú ert.
	mkdir = make directory, gerir nýja möppu.

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone = Afritar geymslu
	git log = Sér allar skráningar
	git status = Shows the working tree status
	git diff = shows changes on commits, commit and working tree etc
	git checkout = switches branches or restore working tree files

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff = Shows changes on commits, comit and working tree etc
	b) git diff --staged = changes between the index and your last commit
	c) git diff commit1 commit2 = compare the version between commits

4. 	Hvað er útgáfustýring (e.version control)?
